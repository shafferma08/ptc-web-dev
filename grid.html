📘 What is CSS Grid?

CSS Grid is a layout system designed for two-dimensional layouts (both rows and columns). It allows you to create complex responsive web designs more easily than older layout techniques like floats or even Flexbox.

✨ Why Use CSS Grid?

  • Ideal for building full page or section-based layouts
  • Handles both columns and rows
  • Reduces the need for deeply nested HTML
  • Pairs well with Flexbox (use Flex for components, Grid for layout)

🧠 Before You Code: Plan Your Grid Layout

Grid is incredibly powerful, but without a plan, it's easy to get lost. Take time to sketch your layout and ask yourself:

  • How many columns and rows will I need?
  • Do any items need to span multiple columns or rows?
  • Am I using grid-template-areas or explicit line-based placement?
Grid starts with the parent container using display: grid. All column, row, and area settings are applied from there.

🪤 Common Grid Mistakes

  • 🟥 Using grid-column or grid-row on the container instead of children
  • 🟥 Forgetting to define both grid-template-columns and grid-template-rows
  • 🟥 Misaligning grid-template-areas with your HTML
  • 🟥 Not using auto-fit or minmax for responsiveness

🔤 Basic Syntax

/* Container */
.parent {
  display: grid;
  grid-template-columns: 1fr 1fr;
  grid-template-rows: auto auto;
  gap: 10px;
}

1fr means “one fraction” of the available space.

🧩 Example: 2x2 Grid

<div class="grid-2x2">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
</div>

.grid-2x2 {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 10px;
}

↔️ Spanning Columns and Rows

.item {
  grid-column: span 2;
  grid-row: span 2;
}

🗺️ Grid Template Areas

.layout {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  grid-template-columns: 200px 1fr;
}

📱 Responsive Grid

.responsive {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}

This automatically adjusts the number of columns based on available space. Super handy for galleries or card layouts!
